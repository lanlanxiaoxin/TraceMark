# TraceMark v0.2.0 发布说明

**发布日期：** 2026-05（W8）  
**定位：** 程序员的日报/周报代笔 + 上下文回忆机（开源 MIT）

## 亮点

- **今日盖章流**：约 5 分钟三步完成主线判断，自动生成日报（模板 / BYOK AI）
- **本周战报 + 周记忆胶囊**：基于已确认资产生成周报，导出 1080×1920 分享图
- **Ctrl+K 回忆机**：FTS 检索工作资产与原始活动，`@last-week` 时间过滤，跳回时间轴高亮
- **界面国际化（中/英）**：设置 → **语言** 切换简体中文 / English；界面、系统通知、托盘与对话框随语言变化
- **报告与复盘双语生成**：日报 / 周报 / 战报 / 周复盘 / 阶段复盘按界面语言选用中英文提示词；AI 产出自动清理异常 Markdown 符号（如 `-*`）
- **首次启动引导**：项目空间 → 监听与隐私 → 上手提示
- **本地埋点看板**：设置页可查看核心行为统计（仅本机）

## 安装

- **Windows**：从 [GitHub Releases](https://github.com/lanlanxiaoxin/TraceMark/releases/tag/v0.2.0) 下载安装包，或源码 `npm run package`
- **开发运行**：`npm install` → `npm run dev`

## 升级说明

- 数据库自动迁移，无需手动操作
- 首次安装会显示 onboarding；老用户若需重看：设置 → 重新运行首次引导

## 已知限制

- 活动采集以 **Windows** 为主；macOS / Linux 仍为预留
- 命令面板回忆命中率依赖本地资产与活动质量，建议完成盖章并确认资产

## 国际化

### 如何切换语言

1. 打开 **设置 → 语言**
2. 选择 **简体中文** 或 **English**
3. 切换后立即生效，偏好保存在本机 `settings.ui_locale`

未手动选择时：浏览器/系统语言以 `en` 开头则默认英文，否则默认中文。

### 覆盖范围

| 范围 | 说明 |
|------|------|
| 界面文案 | 今日、项目、时间轴、报告、设置、命令面板、上传预览等 |
| 主进程 | 日报/周五通知、托盘菜单、文件夹选择与导出对话框 |
| AI 报告 | 日报（含盖章 v3）、周报、本周战报：按语言加载提示词与 system 指令 |
| AI 复盘 | 周复盘、项目阶段复盘：双语提示词；预览排版与报告页一致 |
| 离线复盘 | 五段式结构模板随语言切换（未配置 API 时） |

### 使用提示

- **英文界面**下，输入上下文（资产标题、活动摘要等）仍可能含中文；AI 会翻译为英文输出，请勿期望 100% 自动改写历史中文资产标题
- **离线模式**下报告/复盘使用本地模板，章节结构与语言一致，但部分原始活动句式可能仍较长；建议在「今日」确认资产时使用简短标题
- 安装程序语言（NSIS）与 **应用内界面语言** 独立；应用内语言以设置为准
