# 反馈与试用

感谢使用 TraceMark。本项目面向个人开发者开源，欢迎通过以下渠道反馈。

## GitHub（推荐）

| 用途 | 链接 |
|------|------|
| Bug / 功能请求 | [Issues](https://github.com/lanlanxiaoxin/TraceMark/issues) |
| 问答、试用交流 | [Discussions](https://github.com/lanlanxiaoxin/TraceMark/discussions) |

试用招募请在 Issue 标题注明 `[beta-trial]`，并承诺 **2 周内**回复一条真实使用感受（可很短）。

## 应用内

**设置 → 反馈与社区** 可一键打开上述链接。

## 微信群 / Discord

若尚未建立公开群，可在 Discussions 发帖后由维护者邀请早期试用者。不在仓库内张贴私人联系方式。

## 反馈时请附带

- TraceMark 版本（设置 → 关于）
- Windows 版本
- 复现步骤或截图（请打码敏感窗口标题）
- 是否开启进程监听 / 云端 AI

请勿在 Issue 中粘贴 API Key、完整窗口标题或未脱敏路径。
