# Changelog

All notable changes to TraceMark are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.2.0] - 2026-05-24

### Added

- **Today Seal** three-step flow (mainline → one-liner → evidence) with classic inbox fallback
- Template daily reports (`daily-report-v3`) and BYOK AI enhancement
- **Friday weekly battle** with desktop notification and Markdown / share image export
- **Weekly Memory Capsule** 1080×1920 PNG export
- **Ctrl+K** command palette with FTS5 recall (`@last-week`, `#outcome`) and optional **AI rerank** toggle
- Asset evidence expand and timeline jump
- First-run onboarding (project → privacy → tips)
- Local metrics dashboard and self-check log
- UI i18n (zh / en) for shell, notifications, and AI report prompts

### Changed

- Product branding unified as **TraceMark** (installer, tray, README)
- Command palette: keyword search no longer calls LLM by default; enable **AI rerank** explicitly

### Fixed

- Demo GIF filename aligned with README (`showTraceMark.gif`)
